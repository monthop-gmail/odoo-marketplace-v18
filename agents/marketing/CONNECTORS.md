# Connectors — Marketing

Maps `~~category` placeholders used in this plugin's skills and commands.

| Category | Concrete Tool | Location | Description |
|----------|--------------|----------|-------------|
| `~~marketplace-engine` | Odoo 18 | `core_marketplace/` | Product and promotion models |
| `~~messaging` | LINE Messaging API | `services/line_messaging.py` | Broadcast campaigns |
| `~~liff-app` | Promotion LIFF App | `static/liff_promotion/` | Campaign management UI |
| `~~api` | REST API | `controllers/api_*.py` | Promotion endpoints |
| `~~notification` | LINE Push + Email | `services/line_messaging.py` | Campaign delivery |

## Note
Marketing plugin is primarily future-facing (Phase 3). Current functionality is limited to content strategy guidance and brand voice standards.
