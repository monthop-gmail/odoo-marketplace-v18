# Content Strategy (กลยุทธ์เนื้อหา)

You manage the content strategy that powers the social-driven marketplace. The core principle: 80% value content, 20% selling content.

> If you see unfamiliar `~~category` placeholders, see [CONNECTORS.md](../../CONNECTORS.md).

## The 80/20 Rule

This marketplace is **social-driven** — content attracts community, community drives sales.

### Value Content (80%)

| Type | Purpose | Example |
|------|---------|---------|
| How-To | Teach useful skills | "วิธีเลือกกระเป๋าหนังแท้" |
| Tips & Tricks | Share practical knowledge | "5 เคล็ดลับดูแลรองเท้าหนัง" |
| Behind the Scenes | Build trust & authenticity | "เบื้องหลังการผลิตสินค้า" |
| Community Stories | Feature real customers | "รีวิวจากลูกค้าจริง" |
| Industry News | Position as authority | "เทรนด์แฟชั่นปี 2026" |
| Q&A / Polls | Drive engagement | "คุณชอบสีไหนมากกว่ากัน?" |

### Selling Content (20%)

| Type | Purpose | Example |
|------|---------|---------|
| Product Showcase | Highlight features | "สินค้าใหม่ประจำสัปดาห์" |
| Flash Sales | Create urgency | "ลด 30% วันนี้วันเดียว!" |
| Testimonials | Social proof | "ลูกค้าบอกว่า..." |
| New Arrivals | Drive discovery | "เพิ่งมาถึง! คอลเลคชั่นใหม่" |

## Content Funnel

```
Social Media (YouTube/Facebook/TikTok)
    ↓ Discover (value content attracts)
LINE OA (Follow)
    ↓ Connect (build relationship)
LIFF Mini App (Browse)
    ↓ Shop (natural conversion)
Order + Repeat
    ↓ Advocate (share + review)
```

## Content Guidelines for Sellers
- Lead with value, not with product
- Show the person behind the shop
- Use real photos, not stock images
- Tell stories, not just specifications
- Respond to comments within 2 hours
- Use Thai language naturally (not formal/stiff)

## Metrics to Track

| Metric | Target | Why |
|--------|--------|-----|
| Content engagement rate | >5% | Value resonance |
| LINE follower growth | +10%/month | Community building |
| Content → LIFF click-through | >3% | Funnel effectiveness |
| Repeat visit rate | >40% | Content stickiness |

## Anti-Patterns

| Don't | Do Instead |
|-------|-----------|
| Post only products (100% selling) | Follow 80/20 rule strictly |
| Use pushy sales language | Educate and inspire first |
| Ignore comments/questions | Engage within 2 hours |
| Post the same content everywhere | Adapt format per platform |
| Skip consistency | Post minimum 5x/week |
