# Buyer Journey Analysis - สภา shop

**Created:** 2026-03-03

---

## 1. ช่องทางที่ผู้ซื้อจะเข้ามา

### Primary Channels (LINE-Centric)

```
Social Media Content (YouTube/Facebook/TikTok)
  - Brand Ambassador สร้างคอนเทนต์
  - 80% Entertainment + 20% Selling
  - Product Review/Endorsement
          ↓ Add LINE OA
LINE Official Account
  - Rich Menu → LIFF Marketplace
  - Broadcast Messages
  - Chatbot Consultation
          ↓ Click LIFF
LIFF Mini App (Marketplace)
  - Browse Products
  - Filter Endorsed Products
  - Purchase
```

### Secondary Channels
- **Word of Mouth** — เพื่อนแชร์ LINE OA/Product
- **Ambassador Fan Club** — แฟนคลับตาม Gurus
- **Social Sharing** — แชร์จาก LIFF → Facebook/Twitter
- **QR Code** — ตามร้านค้า/อีเวนต์

## 5. 🔍 Pain Points

### Awareness Stage (ยังไม่รู้จัก)
- ❌ ไม่รู้ว่าสภา shop คืออะไร
- ❌ ไม่เชื่อใจ Brand Ambassador
- ❌ คิดว่าเป็นแค่ Marketplace ทั่วไป

### Consideration Stage (กำลังพิจารณา)
- ❌ เข้าใจยาก ว่าต่างจาก Shopee/Lazada ยังไง
- ❌ ไม่เห็นประโยชน์ของการซื้อผ่าน LINE
- ❌ กังวลเรื่องความน่าเชื่อถือของสินค้า

### Purchase Stage (ตัดสินใจซื้อ)
- ❌ ขั้นตอนการสั่งซื้อยุ่งยาก
- ❌ ไม่มั่นใจในการชำระเงิน
- ❌ กังวลเรื่องการส่งของ/คืนสินค้า

### Retention Stage (กลับมาซื้อซ้ำ)
- ❌ ไม่มีเหตุผลให้กลับมา
- ❌ ลืมว่าเคยซื้ออะไรไป
- ❌ ไม่มีการติดตามหลังการขาย

## 6. 🚀 ข้อเสนอแนะปรับปรุง

### Phase 1: ดึงเข้ามาครั้งแรก

#### 1.1 Leverage Ambassador Fanbase
- แต่ละ Ambassador มีแฟนคลับอยู่แล้ว
- สร้าง "Exclusive Deal" สำหรับแฟนคลับแต่ละคน
- ใช้ LINE OA ของ Ambassador เอง (ถ้ามี)
- Broadcast เมื่อมีสินค้าใหม่ที่ได้รับรอง

#### 1.2 First-Time Incentive
- ส่วนลด 10% สำหรับสั่งซื้อครั้งแรก
- ฟรีค่าจัดส่ง Order แรก
- ของแถมเฉพาะสินค้าที่ Endorse

#### 1.3 Social Proof on Landing
ใน LIFF หน้าแรก แสดง:
- จำนวนสินค้าที่ Endorse แล้ว
- จำนวน Ambassadors
- รีวิวจากผู้ซื้อจริง
- ยอดขายสินค้า populer

---

### Phase 2: ทำให้เข้าใจ Value Proposition

#### 2.1 Clear Messaging
```
❌ "Marketplace สำหรับทุกคน"
✅ "สินค้าคัดสรร โดยกูรูที่ไว้ใจได้"

❌ "ซื้อของผ่าน LINE"
✅ "ช้อปง่าย ใน LINE ไม่ต้องโหลดแอปใหม่"
```

#### 2.2 Onboarding Flow ใน LIFF
1. Welcome Screen → อธิบายสภา shop ใน 1 ประโยค
2. Show Ambassadors → แสดง Gurus ที่มี
3. Show Endorsed Products → สินค้าที่รับรอง
4. First Purchase Guide → สอนสั่งซื้อทีละขั้น

#### 2.3 Trust Signals
- Badge "สภา shop รับรอง" บนสินค้า
- วิดีโอรีวิวจาก Ambassador
- ใบรับรอง/Credentials ของ Gurus
- รีวิวจากผู้ซื้อจริง (พร้อมรูป)

---

### Phase 3: ทำให้กลับมาซื้อซ้ำ

#### 3.1 Personalized Broadcast
ไม่ส่งหว่าน แต่ส่งตามความสนใจ:
- เคยซื้อหมวดไหน → ส่งสินค้าหมวดนั้น
- เคยดูสินค้าไหน → ส่งโปรโมชันสินค้านั้น
- Birthday/Anniversary → ส่วนลดพิเศษ

#### 3.2 Post-Purchase Follow-up
- **Day 1:** ยืนยันการสั่งซื้อ + ติดตามสถานะ
- **Day 3:** แจ้งจัดส่ง + เลข Tracking
- **Day 7:** ถามความพึงพอใจ
- **Day 14:** ขอรีวิว + ส่วนลดครั้งหน้า

#### 3.3 Loyalty Program
- คะแนนสะสมทุกการซื้อ (1 บาท = 1 คะแนน)
- Tier System: Silver/Gold/Platinum
- คะแนนแลกส่วนลด/ของแถม
- Early Access สินค้าใหม่สำหรับสมาชิก

---

### Phase 4: ทำให้แนะนำคนอื่น

#### 4.1 Referral Program
```
"ชวนเพื่อน มาช้อป"
- ผู้ชวน: ได้เครดิต 50 บาท
- ผู้ถูกชวน: ได้ส่วนลด 10%
- ไม่จำกัดจำนวนเพื่อน
```

#### 4.2 Affiliate for Buyers
ผู้ซื้อสามารถเป็น Affiliate ได้:
- แชร์ลิงก์สินค้า
- ได้ Commission 3-5% เมื่อมีคนซื้อ
- ติดตามยอดใน LIFF Dashboard

#### 4.3 Social Sharing Incentive
- แชร์รีวิวบน Facebook → ได้คะแนน
- Tag เพื่อนในคอมเมนต์ → ลุ้นรางวัล
- User-Generated Content → Feature ในเพจ
